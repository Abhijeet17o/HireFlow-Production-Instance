## Quick Start

```bash
# Clone the repository
git clone https://github.com/Abhijeet17o/HireFlow-Production-Instance.git
cd HireFlow-Production-Instance

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

Visit `http://localhost:5173` and click "Continue as Guest" to explore.

## Usage

1. Click **"Continue as Guest"** on the login page
2. Explore the **3 demo campaigns** with sample candidates
3. Create new campaigns or add candidates
4. Drag candidates between pipeline stages
5. All data is saved in your browser's localStorage
